Standard

We want to ensure everything we do is held to the same standard. So we created criteria to help us assess both existing tools and newly developed tools to evaluate openness, accessibility, and more. All tools in our toolbox are graded on a number of criteria against a Green, Amber, Red scale, which will be assessed by members of the link23 community (so get involved today!). Tools with any Red criteria will not be immediately featured on our toolbox however we actively want to work with developers to enable all tools to become Green.

Our standard is meant to evolve and develop over time and we will have a GitHub repository that anyone can comment on and make pull requests to so that we can have ongoing discussions with the community to update this standard. In a future version of this website we will enable live voting and comments on tools in order to rank tools by utility and to allow users to post comments about how they have used the tool in their practice, we will also enable code chunks so tool users can include code snippets demonstrating the tool in practice.

Finally, there are many other standards out there that focus on different aspects of tool development. We will added badges or links to all other standards that any tools our toolbox may conform to, for example if the tool is a recognised by a Digital Public Good or if it adheres to FAIR/FAIR4RS principles.

So without anymore preamble... the link23 standard!
