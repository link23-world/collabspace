link23 is an Open initiative on a mission to improve equity and diversity in the providers, users and beneficiaries of genomic tools and data worldwide. We work across four overlapping pillars focused on:

  1. Curating existing tools
  2. Developing new tools
  3. Running collaborative and competitive challenges
  4. Building an exciting and engaged community all around tools for genomic equity.
